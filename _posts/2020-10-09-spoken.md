음성 데이터(speech data)는 콜센터에서 전화를 받는 인공지능을 만들거나, 구글 어시스턴트같은 음성 기반 인공지능 비서의 알고리즘 등을 개발할때 필요하다.

Datacamp에서 중급자 코스로 Spoken Language Processing in Python 코스가 나왔다. 

이 강좌는 음성 파일을 직접 입출력해서 연구하거나 실제 비지니스용 서비스에 활용하고자 하는 사람들에게 입문용으로 추천한다.

### 1강 Introduction to Spoken Language Processing with Python (무료)
 (파이썬 기반 음성언어 처리 기초)

Python 으로 음성 파일 import하기, sound wave파일 시각화 하기 등을 다룬다. 

### 2강 Using the Python Speech Recognition Library (유료)
(파이썬 기반 음성 인식 라이브러리 활용)

음성 인식API사용하기, 인식기(Recognizer) 다루기, 오디오 파일과 오디오 데이터의 차이 등을 다룬다. 

### 3강 Manipulating Audio Files with PyDub (유료)
(PyDub라이브러리로 음성 파일 다루기)

오디오 파일을 전문적으로 다룰 수 있는 오픈소스, PyDub에 대해 배운다. 오디오 파일의 파라미터, 조작 등에 대해 배운다.

### 4강 Processing text transcribed from spoken Language (유료)
(텍스트로 변환된 음성 언어 처리하기)

텍스트로 변환하는 법, 자연어 전처리 및 감성 분석 등에 적용할 수 있는 방법에 대해 배운다.

Pre-and post-purchase audio snippet transcriptions

아마, 마지막 강좌에서 다룰 실습 코드에 적용할 데이터셋인 것 같다. 

음성 처리는 음성 신호 자체에 대한 이해 뿐만 아니라, 음성 데이터를 불러오고, 전처리하고, 모델에 적용하는 것 까지 알아야 한다. 
즉, 파이썬 같은 프로그래밍 언어를 활용해서 오디오 파일 입출력을 다룰 줄 알아야하고, Scikit-learn같은 기계 학습용 (파이썬)라이브러리를
활용하여 모델 예측/분류 또는 생성 등을 할 줄 알아야한다. 이 코스는 그 기초를 닦기 위해서 가볍게(?) 듣기에 좋아보인다.
